UMsimpleCharts [JQuery Plugin] - DEMO

Usage

Make shure you have embedded jquery.js and jquery.UMsimpleCharts.js 
      
<!DOCTYPE html/>
<html>
 <head>
       <title>your title</title>
       <script type="text/javascript" src="src/jquery.min.js"></script>
       <script type="text/javascript" src="src/jquery.UMsimpleCharts.min.js"></script>
</head>
Then you should instance the UMsimpleChart like in the examples below: 
      
<!DOCTYPE html/>
<html>
 <head>
       <title>your title</title>
       <script type="text/javascript" src="src/jquery.min.js"></script>
       <script type="text/javascript" src="src/jquery.UMsimpleCharts.min.js"></script>
       <script type="text/javascript">
           $('document')ready(function() {
               var progress = $('#idOfDestination').UMsimpleCharts({options});
           });
       </script>
</head>

Options

  • backgroundColor (hexcode) - The desired backgroundcolor of a pie-chart (i.e. '#C7C7C7')
  • size (pixels) - The destination size of the whole chart. Does not include the legend
  • type (string) - Defines the type of the chart. Should be 'pie' or 'bars'
  • legend (boolean) - Defines if a legend will be displayed. Could be 'true' or 'false'
  • values (array of arrays) - The data you would like to be displayed as a chart. Defined in the following format: 
    [['label', percentage, 'color']] // label will be [string], percentage will be [float] between 0 and 100, color will be [hex]
Output Code

simple initialization with no options but values given

                      
  var chart1 = $('#chart1').UMsimpleCharts({values: [['JavaScript', 80, '#FEC501'], ['PHP', 20, '#0233F9']]});
      // default type is 'pie'

initialization with size

                      
  var chart2 = $('#chart2').UMsimpleCharts({size: 130, values: [
               ['JavaScript', 60, '#FEC501'],
               ['PHP', 30, '#0233F9'],
               ['Python', 10, '#BE01BE']
  ]
  });

initialization as bar-chart

                      
  var chart3 = $('#chart3').UMsimpleCharts({ type: 'bars', size: '200', values: [
               ['JavaScript', 60, '#FEC501'],
               ['C++', 40, '#008000']
             ]
  });

initialization as pie-chart without legend

                      
  var chart4 = $('#chart4').UMsimpleCharts({size: 200, type: 'pie', legend: false, values: [
               ['JavaScript', 60, '#FEC501'],
               ['PHP', 30, '#0233F9'],
               ['Python', 10, '#BE01BE']
  ]
  });
